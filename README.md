# Samradh Agarwal - Neural Glass Portfolio (V2)

A high-end, responsive personal portfolio built with Vanilla HTML, CSS, and JS. 
Features a "Bento Box" grid layout, kinetic scroll interactions, and WebGL-style glassmorphism.

## 🚀 Deployment Guide
Since this project uses pure Vanilla technology (no build step, no framework), deployment is instant.

### Option 1: Vercel / Netlify
1.  Drag and drop the project folder into your Vercel or Netlify dashboard.
2.  The site is live.

### Option 2: GitHub Pages
1.  Push this code to a GitHub repository `username/portfolio`.
2.  Enable GitHub Pages in Settings -> Pages -> Source (Main Branch).

## 🛠 Tech Stack
-   **Structure**: Semantic HTML5 with Bento Grid layout.
-   **Style**: Pure CSS Variables, CSS Grid (12-col), Glassmorphism, Noise Textures.
-   **Interaction**: `IntersectionObserver` for scroll reveal, Mouse tracking CSS variables for spotlight effect.
-   **Fonts**: `Clash Display` (Display) & `Satoshi` (Body) via CDN.

## 📂 File Structure
```
/

├── index.html       # Main entry point (includes Templates)

└── script.js        # Logic for Modals, Marquee, & Spotlight
```
